## Run Locally

Clone the project

```bash
  git clone https://github.com/mbayeScientist/fastapi.git
```

Go to the project directory

```bash
  cd my_yaml_course
```

Install dependencies

```bash
  pipenv install
```

Start the server

```bash
  python app.py
```


## Running Tests

To run tests, run the following command

```bash
  pytest . -s -v
```
